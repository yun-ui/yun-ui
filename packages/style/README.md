## 辅助

辅助类提供了几种常见的快速布局方案。

 - 浮动
 - 文字对齐
 - 元素居中

代码如下：

    <p class="left">左浮动</p>
    <p class="right">右浮动</p>
    <p class="text-left">文字左对齐</p>
    <p class="text-center">文字居中对齐</p>
    <p class="text-right">文字右对齐</p>
    <div class="block-center">定宽的块级元素水平居中</div>
